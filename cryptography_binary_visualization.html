Understanding Binary Representation

Binary representation works by assigning each bit a power of 2 based on its position. The rightmost bit is 2^0 (1), the next is 2^1 (2), then 2^2 (4), and so on. For signed integers, the leftmost bit represents the sign (0 for positive, 1 for negative) when using two's complement representation.

1. Positional Values

Each bit position represents a power of 2: 1, 2, 4, 8, 16, 32, etc.

2. Bit Significance

The rightmost bits represent less significant values (lower numbers), while leftmost bits represent more significant values.

3. Sign Representation

In signed integers, the most significant bit indicates positive (0) or negative (1) numbers.

4. Range Limits

Different data types have different ranges: int8 is -128 to 127, int16 is -32,768 to 32,767, etc.

5. Two's Complement

Negative numbers are represented using two's complement, which simplifies arithmetic operations.

Binary Conversion Logic

The binary conversion process involves converting a decimal number to its binary representation. For positive numbers, this is done by repeatedly dividing by 2 and keeping track of the remainders. For negative numbers in signed data types, the two's complement representation is used, which involves inverting all bits and adding 1 to the result.

JavaScript Implementation 
// Function to convert decimal to binary with specified bit length
function decimalToBinary(value, bits) {
    // Handle negative numbers with two's complement
    if (value < 0) {
        // Convert to two's complement
        value = Math.pow(2, bits) + value;
    }
    
    // Convert to binary string
    let binary = value.toString(2);
    
    // Pad with leading zeros to match the required bit length
    while (binary.length < bits) {
        binary = "0" + binary;
    }
    
    return binary;
}

// Function to get bit information
function getBitInfo(binaryString) {
    const bits = [];
    for (let i = 0; i < binaryString.length; i++) {
        bits.push({
            position: binaryString.length - 1 - i, // MSB at position 0
            value: parseInt(binaryString[i])
        });
    }
    return bits;
}
