SHA-1 Algorithm

Hash Function (Deprecated)
Output: 160 bits (40 hex chars)

Algorithm Overview

SHA-1 (Secure Hash Algorithm 1) is a cryptographic hash function that produces a 160-bit (20-byte) hash value. Designed by the NSA, it was widely used for security applications but is now considered cryptographically broken.

The algorithm processes input in 512-bit blocks through 80 rounds of operations using four different logical functions, producing a fixed-size output that was once considered secure for digital signatures and certificates.

160-bit Output

Produces a fixed 160-bit hash value, longer than MD5 but shorter than modern alternatives like SHA-256

Historical Importance

Widely used in TLS/SSL, Git version control, and digital signatures before vulnerabilities were discovered

80 Rounds

Uses 80 processing steps across 4 rounds with different logical functions and constants for each round

Security Status

Collision attacks demonstrated in 2017, making it unsuitable for cryptographic security applications
