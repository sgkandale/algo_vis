Algorithm Overview

Gradient Descent is an optimization algorithm used to minimize a function by iteratively moving in the direction of steepest descent. It is widely used in machine learning to train models by minimizing the cost function (e.g., mean squared error in linear regression).

The algorithm works by computing the gradient (derivative) of the cost function with respect to each parameter and then updating the parameters in the opposite direction of the gradient. The learning rate determines the size of the steps taken during each iteration.

Key Feature

Iteratively minimizes the cost function using gradient information

Learning Method

Moves parameters in direction of steepest descent

Convergence

Converges to local minimum if learning rate is appropriate

Foundation

Core algorithm for training neural networks and other models

Gradient Descent Steps

The Gradient Descent algorithm follows these steps to find the minimum of a function:

1. Initialize Parameters

Set initial values for parameters (θ0, θ1, ...)

2. For Each Iteration

Compute the gradient of the cost function with respect to each parameter

3. Update Parameters

θ = θ - α * ∇J(θ), where α is the learning rate

4. Check Convergence

Continue until convergence criterion is met (max iterations or small gradient)

5. Minimize Cost Function

Repeated updates move parameters toward function minimum

Gradient Descent Algorithm

initialize parameters θ
learning_rate α

repeat until convergence:
    compute gradient: ∇J(θ) = ∂J(θ)/∂θ
    update parameters: θ = θ - α * ∇J(θ)

Implementation

Below is a Python implementation of the Gradient Descent algorithm for minimizing a quadratic cost function. The implementation includes visualization of the cost function and parameter updates over time.

Python Implementation 
def gradient_descent(cost_function, gradient_function, initial_params, learning_rate, 
                       max_iterations=1000, tolerance=1e-6):
    params = initial_params.copy()
    cost_history = []
    
    for i in range(max_iterations):
        cost = cost_function(params)
        cost_history.append(cost)
        
        gradient = gradient_function(params)
        
        # Update parameters
        for j in range(len(params)):
            params[j] -= learning_rate * gradient[j]
        
        # Check convergence
        if abs(gradient[0]) < 1e-3 and abs(gradient[1]) < 1e-3:
            break
    
    return params, cost_history

# Example cost function: f(x,y) = (x-3)^2 + (y-2)^2
def quadratic_cost(params):
    x, y = params
    return (x - 3) ** 2 + (y - 2) ** 2

# Gradient: [df/dx, df/dy] = [2(x-3), 2(y-2)]
def quadratic_gradient(params):
    x, y = params
    return [2 * (x - 3), 2 * (y - 2)]
