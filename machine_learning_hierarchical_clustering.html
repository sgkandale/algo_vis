Hierarchical Clustering

Unsupervised Learning Algorithm
Time Complexity: O(n³)

Algorithm Overview

Hierarchical Clustering is an unsupervised machine learning algorithm that builds a hierarchy of clusters. It creates a tree-like structure of clusters called a dendrogram, which shows the order and distance of merges. Unlike K-Means, hierarchical clustering doesn't require specifying the number of clusters in advance.

The algorithm works by either starting with each point as a cluster and merging the closest pairs (agglomerative), or starting with one cluster and recursively splitting it (divisive). Agglomerative hierarchical clustering is the most common approach and is visualized here.

Key Feature

Builds a hierarchy of clusters without pre-specifying cluster count

Performance

Computationally intensive (O(n³)) but provides rich cluster hierarchy

Space

Requires O(n²) space for distance matrix storage

Approach

Iteratively merges closest clusters until one remains

Algorithm Steps

Agglomerative Hierarchical Clustering follows these iterative steps:

1

Initialize Clusters

Start by treating each data point as an individual cluster.

2

Compute Distance Matrix

Calculate the distance between every pair of clusters using a linkage method.

3

Merge Closest Clusters

Find the two closest clusters and merge them into a new cluster.

4

Update Distance Matrix

Recalculate distances between the new cluster and remaining clusters.

5

Repeat Until Convergence

Continue merging clusters until only one remains.

Pseudocode

function hierarchical_clustering(data, linkage):
    # Step 1: Initialize clusters
    clusters = [each data point as a cluster]
    
    # Step 2: Compute initial distance matrix
    distance_matrix = compute_pairwise_distances(clusters, linkage)
    
    # Initialize dendrogram
    dendrogram = []
    
    while len(clusters) > 1:
        # Step 3: Find closest clusters
        i, j = find_min_distance_indexes(distance_matrix)
        cluster_i = clusters[i]
        cluster_j = clusters[j]
        
        # Step 4: Merge clusters
        new_cluster = merge(cluster_i, cluster_j)
        
        # Update dendrogram
        dendrogram.append({
            'cluster1': cluster_i, 
            'cluster2': cluster_j,
            'distance': distance_matrix[i][j],
            'new_cluster': new_cluster
        })
        
        # Remove old clusters and add new cluster
        clusters = remove_clusters(clusters, i, j)
        clusters.append(new_cluster)
        
        # Step 5: Update distance matrix
        distance_matrix = update_distance_matrix(distance_matrix, i, j, clusters, linkage)
    
    return dendrogram

Implementation

Below is a Python implementation of Agglomerative Hierarchical Clustering. The implementation includes different linkage methods (single, complete, average) and builds a dendrogram to visualize the hierarchy.

Python Implementation 
import numpy as np
from scipy.spatial.distance import pdist, squareform
from scipy.cluster.hierarchy import dendrogram, linkage
import matplotlib.pyplot as plt

class HierarchicalClustering:
    def __init__(self, linkage_method='average'):
        self.linkage_method = linkage_method
        self.dendrogram = None
        
    def fit(self, X):
        # Compute pairwise distances
        dist_matrix = pdist(X, 'euclidean')
        
        # Perform hierarchical clustering
        linkage_matrix = linkage(dist_matrix, method=self.linkage_method)
        
        # Store results for dendrogram
        self.dendrogram = linkage_matrix
        
        return self
    
    def plot_dendrogram(self):
        if self.dendrogram is None:
            raise ValueError("Model has not been fitted yet.")
            
        plt.figure(figsize=(12, 6))
        dendrogram(self.dendrogram)
        plt.title(f'Dendrogram (Linkage: {self.linkage_method.capitalize()})')
        plt.xlabel('Data Points')
        plt.ylabel('Distance')
        plt.show()

# Example usage
from sklearn.datasets import make_blobs

# Generate sample data
X, y = make_blobs(n_samples=50, centers=3, n_features=2, random_state=42)

# Create and fit hierarchical clustering model
hc_single = HierarchicalClustering(linkage_method='single')
hc_single.fit(X)

hc_complete = HierarchicalClustering(linkage_method='complete')
hc_complete.fit(X)

hc_average = HierarchicalClustering(linkage_method='average')
hc_average.fit(X)

# Plot dendrograms
hc_single.plot_dendrogram()
hc_complete.plot_dendrogram()
hc_average.plot_dendrogram()

Applications

Hierarchical Clustering is widely used in various domains:

  • Phylogenetic tree construction in biology
  • Social network analysis and community detection
  • Document clustering and topic hierarchy
  • Image segmentation in computer vision
  • Market segmentation and customer profiling

Advantages

  • Doesn't require pre-specifying number of clusters
  • Provides intuitive hierarchical structure
  • Dendrogram visualization helps in interpretation
  • Works with any distance metric
  • Flexible with different linkage methods

Limitations

  • Computationally expensive for large datasets
  • Sensitive to noise and outliers
  • Different linkage methods yield different results
  • Once clusters are merged, they cannot be split
  • Difficult to scale to large datasets

Linkage Methods

The distance between clusters can be defined in different ways:

  • Single Linkage: Minimum distance between clusters
  • Complete Linkage: Maximum distance between clusters
  • Average Linkage: Average distance between clusters
  • Ward's Method: Minimizes within-cluster variance
