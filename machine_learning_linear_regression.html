Linear Regression

Supervised Learning Algorithm
Type: Regression

Algorithm Overview

Linear Regression is a fundamental supervised learning algorithm used for predicting continuous numerical values based on input features. It models the relationship between a dependent variable and one or more independent variables by fitting a linear equation to observed data.

The algorithm works by finding the best-fitting straight line through the data points that minimizes the sum of squared residuals (differences between predicted and actual values). This line can then be used to make predictions on new, unseen data.

Objective

Minimize the sum of squared differences between observed and predicted values

Equation

y = β₀ + β₁x + ε (simple linear regression with one feature)

Assumptions

Linearity, independence, homoscedasticity, and normality of residuals

Applications

Economics, finance, environmental modeling, and any domain with continuous outcomes

Algorithm Steps

Linear Regression involves the following steps to create a predictive model:

1. Data Collection

Gather a dataset with input features and corresponding target values

2. Model Representation

Define the hypothesis function: h(x) = β₀ + β₁x for simple linear regression

3. Cost Function

Define the cost function: J(β₀, β₁) = (1/2m) * Σ(h(xⁱ) - yⁱ)²

4. Optimization

Minimize the cost function using gradient descent or normal equation

5. Model Evaluation

Evaluate model performance using metrics like R², MSE, or RMSE

6. Prediction

Use the trained model to make predictions on new data

Applications

Linear Regression is widely used across various domains for predictive modeling:

  • Economics: Predicting GDP growth, inflation rates
  • Real Estate: Estimating house prices based on features
  • Healthcare: Predicting patient outcomes based on biomarkers
  • Marketing: Forecasting sales based on advertising spend
  • Climate Science: Modeling temperature changes over time
  • Finance: Stock price prediction and risk assessment

# Gradient Descent for Linear Regression
def gradient_descent(X, y, learning_rate, iterations):
    m = len(y)  # Number of training examples
    β = [0, 0]   # Initialize parameters [β₀, β₁]
    
    for i in range(iterations):
        h = β[0] + β[1] * X  # Hypothesis
        error = h - y
        
        # Compute gradients
        grad0 = (1/m) * np.sum(error)
        grad1 = (1/m) * np.sum(error * X)
        
        # Update parameters
        β[0] = β[0] - learning_rate * grad0
        β[1] = β[1] - learning_rate * grad1
        
        # Compute cost (optional)
        cost = (1/(2*m)) * np.sum(error ** 2)
    
    return β

# Normal Equation (Closed-form solution)
def normal_equation(X, y):
    X_b = np.c_[np.ones((len(X), 1)), X]  # Add x0 = 1 to each instance
    β = np.linalg.inv(X_b.T.dot(X_b)).dot(X_b.T).dot(y)
    return β

Evaluation Metrics

Key metrics for evaluating linear regression models:

  • R² (R-squared): Proportion of variance explained (0-1, higher is better)
  • MSE (Mean Squared Error): Average squared difference (lower is better)
  • RMSE (Root MSE): Square root of MSE (in same units as target)
  • MAE (Mean Absolute Error): Average absolute difference (robust to outliers)
