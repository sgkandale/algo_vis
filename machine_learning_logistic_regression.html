Logistic Regression

Classification Algorithm
Type: Binary Classification

Algorithm Overview

Logistic Regression is a fundamental classification algorithm used for predicting binary outcomes (0 or 1) based on input features. Despite its name, it's a classification algorithm that models the probability of a binary outcome using a logistic function.

The algorithm works by fitting a sigmoid curve to the data that maps any real-valued number to a value between 0 and 1, which can be interpreted as a probability. This probability is then thresholded to make class predictions.

Objective

Model the probability of class membership using a logistic function

Sigmoid Function

σ(z) = 1 / (1 + e⁻ᶻ) transforms linear output to probability

Decision Boundary

Defined by θ₀ + θ₁x₁ + θ₂x₂ = 0, separating classes

Applications

Medical diagnosis, spam detection, credit scoring, and more

Algorithm Steps

Logistic Regression involves the following steps to create a classification model:

1. Data Preparation

Prepare labeled dataset with features and binary target variable

2. Hypothesis Function

Define h(x) = σ(θᵀx) where σ is the sigmoid function

3. Cost Function

Define log loss: J(θ) = -1/m Σ [yⁱ log(h(xⁱ)) + (1-yⁱ) log(1-h(xⁱ))]

4. Optimization

Minimize cost using gradient descent to find optimal θ

5. Decision Boundary

Set threshold (usually 0.5) to classify points based on probability

6. Prediction

Use trained model to predict class probabilities for new data

Applications

Pseudocode

# Sigmoid function
def sigmoid(z):
    return 1 / (1 + exp(-z))

# Logistic Regression with Gradient Descent
def logistic_regression(X, y, learning_rate, iterations):
    m, n = X.shape
    θ = zeros(n)  # Initialize parameters
    
    for i in range(iterations):
        # Compute hypothesis and error
        z = dot(X, θ)
        h = sigmoid(z)
        error = h - y
        
        # Compute gradient
        gradient = dot(X.T, error) / m
        
        # Update parameters
        θ = θ - learning_rate * gradient
        
        # Compute cost
        cost = (-1/m) * sum(y * log(h) + (1-y) * log(1-h))
    
    return θ

# Predict function
def predict(X, θ, threshold=0.5):
    probabilities = sigmoid(dot(X, θ))
    return (probabilities >= threshold).astype(int)

Evaluation Metrics

Key metrics for evaluating logistic regression models:

  • Accuracy: Proportion of correct predictions
  • Precision: Proportion of true positives among positive predictions
  • Recall: Proportion of positives correctly identified
  • F1 Score: Harmonic mean of precision and recall
  • ROC-AUC: Area under the Receiver Operating Characteristic curve
