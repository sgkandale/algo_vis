Principal Component Analysis

Dimensionality Reduction Technique
Time Complexity: O(p³ + n×p²)

Algorithm Overview

Principal Component Analysis (PCA) is a statistical technique for dimensionality reduction that transforms a set of possibly correlated variables into a set of linearly uncorrelated variables called principal components. The first principal component accounts for the largest possible variance in the data, with each succeeding component accounting for the highest remaining variance under the constraint of orthogonality to the preceding components.

PCA is widely used for data compression, feature extraction, and visualization of high-dimensional data. It helps to reduce the dimensionality of data while preserving as much variance as possible.

Key Feature

Transforms correlated variables into orthogonal principal components

Performance

Efficient for dimensionality reduction on large datasets

Space

Requires O(p²) space for covariance matrix

Approach

Eigenvalue decomposition of the covariance matrix

Algorithm Steps

PCA transforms data through a series of mathematical steps:

1

Standardize the Data

Center the data by subtracting the mean of each feature, and scale to unit variance.

2

Compute Covariance Matrix

Calculate the covariance matrix of the standardized data to understand feature relationships.

3

Eigenvalue Decomposition

Compute the eigenvalues and eigenvectors of the covariance matrix.

4

Sort Eigenvectors

Sort eigenvectors by their eigenvalues in descending order to prioritize principal components.

5

Project Data

Transform the original data using the selected eigenvectors to get the principal components.

Pseudocode

function PCA(data, n_components):
    # Step 1: Standardize the data
    standardized = (data - mean(data)) / standard_deviation(data)
    
    # Step 2: Compute covariance matrix
    covariance = cov(standardized)
    
    # Step 3: Eigenvalue decomposition
    eigenvalues, eigenvectors = eig(covariance)
    
    # Step 4: Sort eigenvectors by eigenvalues (descending)
    sorted_indices = argsort(eigenvalues)[::-1]
    eigenvalues = eigenvalues[sorted_indices]
    eigenvectors = eigenvectors[:, sorted_indices]
    
    # Step 5: Select top n_components eigenvectors
    principal_components = eigenvectors[:, :n_components]
    
    # Transform data
    transformed_data = dot(standardized, principal_components)
    
    return transformed_data, eigenvalues, eigenvectors

Implementation

Below is a Python implementation of PCA using NumPy. The implementation follows the standard PCA steps of standardization, covariance matrix calculation, eigenvalue decomposition, and data transformation.

Python Implementation 
import numpy as np

class PCA:
    def __init__(self, n_components=2):
        self.n_components = n_components
        self.components = None
        self.mean = None
        self.explained_variance = None
        
    def fit(self, X):
        # Step 1: Standardize the data (center and scale)
        self.mean = np.mean(X, axis=0)
        X_centered = X - self.mean
        
        # Step 2: Compute covariance matrix
        cov_matrix = np.cov(X_centered, rowvar=False)
        
        # Step 3: Eigenvalue decomposition
        eigenvalues, eigenvectors = np.linalg.eigh(cov_matrix)
        
        # Step 4: Sort eigenvectors by eigenvalues (descending)
        sorted_indices = np.argsort(eigenvalues)[::-1]
        eigenvalues = eigenvalues[sorted_indices]
        eigenvectors = eigenvectors[:, sorted_indices]
        
        # Step 5: Store the top components
        self.components = eigenvectors[:, :self.n_components]
        self.explained_variance = eigenvalues[:self.n_components]
        self.explained_variance_ratio = eigenvalues / np.sum(eigenvalues)
        
        return self
    
    def transform(self, X):
        X_centered = X - self.mean
        return np.dot(X_centered, self.components)
    
    def fit_transform(self, X):
        self.fit(X)
        return self.transform(X)

Applications

PCA is widely used in various domains for data analysis:

  • Data compression and storage optimization
  • Feature extraction for machine learning
  • Visualization of high-dimensional data
  • Noise reduction in signal processing
  • Genomics and bioinformatics data analysis

Advantages

  • Reduces dimensionality while preserving variance
  • Removes correlated features
  • Improves algorithm performance and efficiency
  • Helps visualize high-dimensional data
  • Provides interpretable components

Limitations

  • Sensitive to feature scaling
  • Assumes linear relationships between variables
  • Principal components can be difficult to interpret
  • May discard important variance in the data
  • Computationally expensive for high-dimensional data

Explained Variance

The explained variance ratio indicates how much information (variance) each principal component contributes:

  • PC1: The first principal component captures the most variance
  • PC2: The second principal component captures the next most variance
  • Cumulative Variance: Sum of explained variances from all components
