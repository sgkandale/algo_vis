How HyperLogLog Works:

1. Hash: Convert each element to a binary hash

2. Bucket: Use first b bits to determine which bucket (0-15 for b=4)

3. Count Zeros: Count leading zeros in remaining bits

4. Update: Keep maximum zero count per bucket

5. Estimate: Use harmonic mean to estimate cardinality
