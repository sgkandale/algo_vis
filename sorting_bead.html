Bead Sort

Gravity Sorting Algorithm
Time Complexity: O(S) where S is sum of input

Algorithm Overview

Bead Sort (also known as Gravity Sort) is a natural sorting algorithm that mimics beads falling under gravity. It works by representing positive integers as beads on vertical rods. When the beads are allowed to fall, they arrange themselves in sorted order due to gravity.

This algorithm is unique because it doesn't compare elements. Instead, it leverages the physical properties of gravity to sort numbers. Bead sort is efficient for sorting lists of positive integers, especially when the integers are small, but becomes impractical for large numbers.

Key Feature

Non-comparison based algorithm that uses gravity

Performance

O(S) where S is the sum of the input integers

Space

O(n * max) where n is count, max is largest number

Approach

Physical simulation of beads falling on vertical rods

Algorithm Steps

Bead Sort processes data through these physical simulation steps:

1. Setup Rods

Create vertical rods representing each number in the array. Place beads on each rod according to the number value.

2. Release Beads

Release the beads and let them fall under gravity. Beads fall to the lowest available position on their rod.

3. Collect Results

After all beads settle, count the beads on each rod from bottom to top to get the sorted array.

Pseudocode

function beadSort(arr):
    n = length(arr)
    if n <= 1: return arr
    
    max_val = max(arr)
    
    # Create bead matrix
    beads = matrix of zeros with dimensions [max_val, n]
    
    # Place beads
    for i from 0 to n-1:
        for j from 0 to arr[i]-1:
            beads[j][i] = 1
    
    # Let beads fall
    for col from 0 to n-1:
        beads_in_col = 0
        # Count beads in column
        for row from 0 to max_val-1:
            if beads[row][col] == 1:
                beads_in_col += 1
        # Drop beads to bottom
        for row from max_val-1 downto 0:
            if beads_in_col > 0:
                beads[row][col] = 1
                beads_in_col -= 1
            else:
                beads[row][col] = 0
    
    # Convert beads back to numbers
    sorted_arr = []
    for row from 0 to max_val-1:
        count = 0
        for col from 0 to n-1:
            if beads[row][col] == 1:
                count += 1
        sorted_arr.append(count)
    
    return sorted_arr

Implementation

Below is a Python implementation of the Bead Sort algorithm. The algorithm creates a matrix representation of beads, then simulates the falling process before converting the settled beads back into sorted numbers.

Python Implementation 
def bead_sort(arr):
    if not arr:
        return []
    
    # Find maximum value to determine matrix height
    max_val = max(arr)
    
    # Create bead matrix (2D list)
    beads = [[0] * len(arr) for _ in range(max_val)]
    
    # Place beads
    for i, num in enumerate(arr):
        for j in range(num):
            beads[j][i] = 1
            
    # Let beads fall
    for col in range(len(arr)):
        beads_in_col = 0
        # Count beads in current column
        for row in range(max_val):
            beads_in_col += beads[row][col]
        
        # Reset column and drop beads to bottom
        for row in range(max_val):
            if max_val - row <= beads_in_col:
                beads[row][col] = 1
            else:
                beads[row][col] = 0
    
    # Convert fallen beads back to numbers
    sorted_arr = []
    for row in range(max_val):
        count = sum(beads[row])
        if count == 0:
            break
        sorted_arr.append(count)
    
    # Return in ascending order
    return sorted_arr

# Example usage
arr = [3, 1, 4, 2]
sorted_arr = bead_sort(arr)
print("Original:", arr)
print("Sorted:", sorted_arr)

Applications

Bead Sort has niche applications where its unique properties are advantageous:

  • Educational demonstrations of non-comparison sorting
  • Sorting positive integers in hardware implementations
  • Physical computing and mechanical sorting devices
