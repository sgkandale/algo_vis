Bubble Sort

Sorting Algorithm
Time Complexity: O(n²)

Algorithm Overview

Bubble Sort is a simple, comparison-based sorting algorithm that repeatedly steps through a list, compares adjacent elements, and swaps them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, indicating the list is sorted.

Despite its simplicity, Bubble Sort is not efficient for large datasets. Its name comes from the way smaller elements "bubble" to the top of the list as the algorithm progresses. While not practical for most real-world applications, it serves as an excellent educational tool for understanding sorting algorithms.

Key Feature

Simple implementation with minimal code

O(n²) time complexity in worst and average cases

Algorithm Steps

Bubble Sort involves the following steps to sort an array:

1. Start Pass

Begin a new pass through the array, starting from the first element

2. Compare Adjacent

Compare each pair of adjacent elements from the beginning of the array

3. Swap if Needed

If elements are in the wrong order, swap them

4. Complete Pass

After each pass, the largest unsorted element bubbles to the end

5. Repeat

Repeat passes until no swaps are needed

Pseudocode

procedure bubbleSort(array)
    n = length(array)
    swapped = true
    
    while swapped
        swapped = false
        for i from 1 to n-1
            if array[i-1] > array[i]
                swap(array[i-1], array[i])
                swapped = true
            end if
        end for
        n = n - 1  // Last element is sorted
    end while
end procedure

Implementation

Below is a Python implementation of the Bubble Sort algorithm. The algorithm repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

Python Implementation 
def bubble_sort(arr):
    # Length of the array
    n = len(arr)
    
    # Traverse through all array elements
    for i in range(n):
        swapped = False
        
        # Last i elements are already in place
        for j in range(0, n-i-1):
            # Traverse the array from 0 to n-i-1
            # Swap if the element found is greater than the next
            if arr[j] > arr[j+1]:
                # Swap elements
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
                
        # If no swaps occurred, array is sorted
        if not swapped:
            break
            
    return arr

# Example usage
arr = [64, 34, 25, 12, 22, 11, 90]
sorted_arr = bubble_sort(arr)
print("Sorted array:", sorted_arr)

Applications

While not efficient for large datasets, Bubble Sort has educational and specialized applications:

  • Teaching introductory computer science concepts
  • Sorting small datasets where simplicity is preferred
  • Detecting nearly sorted arrays efficiently (best case O(n))
  • Understanding algorithm optimization techniques
  • Embedded systems with memory constraints

Complexity Analysis

Bubble Sort's time complexity varies based on the input:

Best Case: O(n)

When the array is already sorted, only one pass is needed

Average Case: O(n²)

For randomly ordered arrays, requires multiple passes

Worst Case: O(n²)

When the array is reverse sorted, maximum comparisons needed
