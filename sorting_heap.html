Heap Sort

Sorting Algorithm
Time Complexity: O(n log n)

Algorithm Overview

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure. It works by first building a max-heap from the input data, then repeatedly extracting the maximum element from the heap and rebuilding the heap until all elements are sorted.

Heap Sort is an in-place algorithm with O(n log n) time complexity in all cases, making it efficient for large datasets. It's not stable but has the advantage of a constant space complexity, making it useful in memory-constrained environments.

Algorithm Steps

Heap Sort involves the following steps to sort an array:

1. Build Max Heap

Transform the array into a max-heap where parent nodes are always greater than children

2. Swap Root

Swap the root (largest element) with the last element of the heap

3. Reduce Heap

Reduce heap size by one (excluding the sorted elements)

4. Heapify

Heapify the root to maintain the max-heap property

5. Repeat

Repeat steps 2-4 until the heap is empty

Pseudocode

procedure heapSort(array)
    n = length(array)
    
    // Build max heap
    for i from floor(n/2) down to 0
        heapify(array, n, i)
    
    // Extract elements from heap
    for i from n-1 down to 0
        swap array[0] and array[i]  // Move root to end
        heapify(array, i, 0)        // Heapify reduced heap

procedure heapify(array, n, i)
    largest = i
    left = 2*i + 1
    right = 2*i + 2
    
    // If left child is larger than root
    if left < n and array[left] > array[largest]
        largest = left
    
    // If right child is larger than largest
    if right < n and array[right] > array[largest]
        largest = right
    
    // If largest is not root
    if largest != i
        swap array[i] and array[largest]
        heapify(array, n, largest)  // Recursively heapify affected subtree

Implementation

Below is a Python implementation of the Heap Sort algorithm. The algorithm consists of the main heap_sort() function and a heapify() function that maintains the heap property.

Python Implementation 
def heap_sort(arr):
    n = len(arr)
    
    # Build max heap
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    
    # Extract elements one by one
    for i in range(n-1, 0, -1):
        # Swap root with last element
        arr[0], arr[i] = arr[i], arr[0]
        # Heapify the reduced heap
        heapify(arr, i, 0)

def heapify(arr, n, i):
    largest = i  # Initialize largest as root
    left = 2 * i + 1
    right = 2 * i + 2
    
    # Check if left child exists and is greater than root
    if left < n and arr[left] > arr[largest]:
        largest = left
    
    # Check if right child exists and is greater than largest
    if right < n and arr[right] > arr[largest]:
        largest = right
    
    # Change root if needed
    if largest != i:
        # Swap
        arr[i], arr[largest] = arr[largest], arr[i]
        # Recursively heapify the affected subtree
        heapify(arr, n, largest)

# Example usage
arr = [12, 11, 13, 5, 6, 7]
heap_sort(arr)
print("Sorted array:", arr)

Applications

Heap Sort is used in various scenarios where guaranteed performance is required:

  • Systems requiring predictable performance (real-time systems)
  • Memory-constrained environments (in-place sorting)
  • Implementing priority queues
  • External sorting algorithms
  • K-way merge algorithms

Complexity Analysis
