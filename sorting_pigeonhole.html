Pigeonhole Sort

Sorting Algorithm
Time Complexity: O(n + range)

Algorithm Overview

Pigeonhole Sort is a non-comparison sorting algorithm that works by placing each element into its corresponding "pigeonhole". It's particularly efficient for sorting integers where the range of possible values (N) is approximately the same as the number of elements (n).

The algorithm works by finding the minimum and maximum values to determine the range, creating an array of pigeonholes (lists), distributing elements into the pigeonholes, and then gathering the elements in order.

Key Feature

Non-comparison based sorting with linear time complexity

Performance

O(n + N) time complexity where N is the range of values

Space

O(n + N) additional space for pigeonholes

Approach

Distribute elements into pigeonholes then gather in order

Algorithm Steps

Pigeonhole Sort processes the array in four distinct phases:

1. Find Range

Determine the minimum and maximum values to calculate the range

2. Create Pigeonholes

Create an array of empty pigeonholes (lists) with size = range + 1

3. Distribute Elements

Place each element in the pigeonhole corresponding to its value

4. Gather Elements

Iterate through pigeonholes in order and place elements back in the array

Pseudocode

function pigeonholeSort(array)
    // Find min and max values
    min = min(array)
    max = max(array)
    range = max - min + 1
    
    // Create pigeonholes
    pigeonholes = array of size range with empty lists
    
    // Put elements in pigeonholes
    for each element in array
        index = element - min
        append element to pigeonholes[index]
    
    // Gather elements back into array
    index = 0
    for i from 0 to range-1
        for each element in pigeonholes[i]
            array[index] = element
            index = index + 1
    
    return array

Implementation

Below is a Python implementation of the Pigeonhole Sort algorithm. Note that this algorithm is most efficient when the range of values is approximately equal to the number of elements.

Python Implementation 
def pigeonhole_sort(arr):
    # Find min and max values in the array
    min_val = min(arr)
    max_val = max(arr)
    size = max_val - min_val + 1
    
    # Create pigeonholes
    pigeonholes = [[] for _ in range(size)]
    
    # Distribute elements into pigeonholes
    for num in arr:
        index = num - min_val
        pigeonholes[index].append(num)
    
    # Gather elements from pigeonholes
    index = 0
    for hole in pigeonholes:
        for num in hole:
            arr[index] = num
            index += 1
    
    return arr

# Example usage
arr = [8, 3, 2, 7, 4, 9, 1, 6, 5]
sorted_arr = pigeonhole_sort(arr)
print("Sorted array:", sorted_arr)

Applications

Pigeonhole Sort is particularly useful in specific scenarios:

  • Sorting integers with a limited range of values
  • Sorting data where the range of values is known
  • Situations where stability is required
  • Sorting large datasets with a small range of values
  • Educational purposes to demonstrate non-comparison sorting

Complexity Analysis

Pigeonhole Sort has a time complexity of O(n + N) where n is the number of elements and N is the range of key values. The algorithm is efficient only when N is approximately the same magnitude as n.

The space complexity is O(n + N) due to the need to create pigeonholes for each possible value in the range. This makes it impractical for large ranges but very efficient for small ranges.

Limitations

Pigeonhole Sort requires additional space proportional to the range of input values. This makes it inefficient for sorting large integers or floating-point numbers.

The algorithm only works for integers or discrete values that can be mapped to integers. It cannot be used for strings or other non-numeric data without modification.
