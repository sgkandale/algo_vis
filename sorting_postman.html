Postman Sort

Distribution Sorting Algorithm
Time Complexity: O(n + k)

Algorithm Overview

Postman Sort (also known as Bucket Sort) is a distribution sorting algorithm that works by distributing elements into a number of buckets. Each bucket is then sorted individually, either using a different sorting algorithm or by recursively applying the bucket sort.

This algorithm is efficient when the input is uniformly distributed over a range. It works by creating buckets for different ranges and distributing elements into these buckets. The buckets are then sorted individually and concatenated to produce the sorted array.

Key Feature

Distributes elements into buckets based on their value

Performance

Efficient when input is uniformly distributed over a range

Space

O(n + k) where k is the number of buckets

Approach

Bucket distribution followed by individual sorting

Algorithm Steps

Postman Sort processes data through these distinct phases:

1. Determine Range

Find the minimum and maximum values to determine the range of the input data.

2. Create Buckets

Create a number of empty buckets based on the range and distribution of the data.

3. Distribute Elements

Scatter the elements into their respective buckets based on their value.

4. Sort Buckets

Sort each bucket individually using a stable sorting algorithm like insertion sort.

5. Concatenate Buckets

Gather all sorted buckets to form the final sorted array.

Pseudocode

function postmanSort(arr, bucketCount):
    n = length(arr)
    if n <= 1: return arr
    
    // Step 1: Find min and max
    minVal = min(arr)
    maxVal = max(arr)
    
    // Step 2: Create buckets
    buckets = array of bucketCount empty lists
    range = (maxVal - minVal) / bucketCount
    
    // Step 3: Distribute elements into buckets
    for each element in arr:
        bucketIndex = floor((element - minVal) / range)
        if bucketIndex >= bucketCount:
            bucketIndex = bucketCount - 1
        append element to buckets[bucketIndex]
    
    // Step 4: Sort each bucket
    for each bucket in buckets:
        sort(bucket)  // using insertion sort or other stable sort
    
    // Step 5: Concatenate buckets
    sortedArr = []
    for each bucket in buckets:
        append bucket to sortedArr
    
    return sortedArr

Implementation

Below is a Python implementation of the Postman Sort algorithm. The algorithm efficiently distributes elements into buckets and then sorts each bucket using insertion sort.

Python Implementation 
def postman_sort(arr, bucket_count=5):
    if len(arr) <= 1:
        return arr
    
    # Find min and max values
    min_val = min(arr)
    max_val = max(arr)
    
    # Calculate range per bucket
    val_range = max_val - min_val
    if val_range == 0:
        return arr
    
    # Create empty buckets
    buckets = [[] for _ in range(bucket_count)]
    
    # Distribute elements into buckets
    for num in arr:
        bucket_idx = int(bucket_count * (num - min_val) / val_range)
        if bucket_idx == bucket_count:
            bucket_idx = bucket_count - 1
        buckets[bucket_idx].append(num)
    
    # Sort each bucket using insertion sort
    for i in range(bucket_count):
        insertion_sort(buckets[i])
    
    # Concatenate buckets
    sorted_arr = []
    for bucket in buckets:
        sorted_arr.extend(bucket)
    
    return sorted_arr

def insertion_sort(bucket):
    for i in range(1, len(bucket)):
        key = bucket[i]
        j = i - 1
        while j >= 0 and bucket[j] > key:
            bucket[j+1] = bucket[j]
            j -= 1
        bucket[j+1] = key

# Example usage
arr = [38, 27, 43, 3, 9, 82, 10]
sorted_arr = postman_sort(arr)
print("Original:", arr)
print("Sorted:", sorted_arr)

Applications

Postman Sort is particularly effective in scenarios with uniformly distributed data:

  • Sorting floating-point numbers in a known range
  • Sorting uniformly distributed integer data
  • Sorting data that can be partitioned into categories
  • Parallel processing of large datasets
  • External sorting algorithms
