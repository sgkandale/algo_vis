Selection Sort

Sorting Algorithm
Time Complexity: O(n²)

Algorithm Overview

Selection Sort is a simple, in-place comparison-based sorting algorithm. It works by dividing the input list into two parts: a sorted sublist which is built from left to right, and the remaining unsorted sublist. The algorithm repeatedly selects the smallest (or largest) element from the unsorted sublist and moves it to the end of the sorted sublist.

While not efficient for large datasets, Selection Sort has the advantage of making the minimum number of swaps (only O(n) swaps). This makes it useful in situations where write operations are expensive. The algorithm is easy to understand and implement, making it a good introductory sorting algorithm.

Key Feature

Minimizes the number of swaps (only O(n) swaps)

Performance

O(n²) time complexity in all cases (best, average, worst)

Space

In-place sorting with O(1) additional space

Approach

Brute force method with two nested loops

Algorithm Steps

Selection Sort involves the following steps to sort an array:

1. Initialization

Start with the first element as the starting point. The sorted sublist is initially empty.

2. Find Minimum

Traverse the unsorted sublist to find the smallest element. Initialize the current element as the minimum and compare with each subsequent element.

3. Swap Elements

Swap the found minimum element with the first element of the unsorted sublist. This moves the smallest element to the end of the sorted sublist.

4. Move Boundary

Move the boundary between the sorted and unsorted sublists one element to the right. Repeat steps 2-4 until the entire list is sorted.

Pseudocode

procedure selectionSort(array)
    n = length(array)
    
    for i from 0 to n-2
        minIndex = i
        
        for j from i+1 to n-1
            if array[j] < array[minIndex]
                minIndex = j
            end if
        end for
        
        if minIndex != i
            swap array[i] and array[minIndex]
        end if
    end for
end procedure

Implementation

Below is a Python implementation of the Selection Sort algorithm. The algorithm consists of two nested loops: the outer loop tracks the boundary between sorted and unsorted portions, while the inner loop finds the smallest element in the unsorted portion.

Python Implementation 
def selection_sort(arr):
    # Traverse through all array elements
    for i in range(len(arr)):
        # Find the minimum element in remaining unsorted array
        min_idx = i
        for j in range(i+1, len(arr)):
            if arr[j] < arr[min_idx]:
                min_idx = j
                
        # Swap the found minimum element with the first element
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
        
    return arr

# Example usage
arr = [64, 25, 12, 22, 11]
sorted_arr = selection_sort(arr)
print("Sorted array:", sorted_arr)

Applications

Selection Sort is used in various scenarios where simplicity is more important than efficiency:

  • Educational purposes to teach sorting fundamentals
  • Sorting small datasets where O(n²) is acceptable
  • Situations with limited memory (in-place sorting)
  • When write operations are expensive (minimal swaps)
  • Embedded systems with constrained resources
