Strand Sort

Sorting Algorithm
Time Complexity: O(n) to O(n²)

Algorithm Overview

Strand Sort is a recursive sorting algorithm that works by repeatedly extracting sorted subsequences (strands) from the unsorted list and merging them into a sorted list. It's particularly efficient for linked lists but can also be applied to arrays.

The algorithm gets its name from the way it extracts "strands" of sorted elements. It has two main phases: extracting sorted strands from the input list and merging these strands into the output list. This process continues until the input list is empty.

Key Feature

Extracts sorted strands and merges them recursively

Performance

O(n) in best case (already sorted), O(n²) average and worst case

Space

O(n) additional space for strands and merging

Approach

Recursive strand extraction and merging

Algorithm Steps

Strand Sort works through two main phases: strand extraction and merging:

1. Strand Extraction

Start with the first element as the initial strand. Sequentially find elements that are larger than the last element in the strand to extend it.

2. Remove Strand

Remove the extracted strand from the input list, leaving the remaining elements.

3. Merge Strand

Merge the newly extracted strand with the output list (which is built from previously extracted strands) in sorted order.

4. Repeat

Continue extracting strands from the remaining input list until it is empty.

5. Completion

When the input list is empty, the output list contains the fully sorted result.

Pseudocode

procedure strandSort(inputList)
    outputList = empty list
    
    while inputList is not empty
        // Create a new strand
        sublist = [inputList[0]]
        remove inputList[0]
        i = 0
        
        // Build the strand
        while i < length(inputList)
            if inputList[i] > last element of sublist
                append inputList[i] to sublist
                remove inputList[i]
            else
                i = i + 1
        
        // Merge the strand into output
        outputList = merge(outputList, sublist)
    
    return outputList

procedure merge(list1, list2)
    result = empty list
    while both lists are not empty
        if first element of list1 < first element of list2
            append list1[0] to result and remove it
        else
            append list2[0] to result and remove it
    append remaining elements of list1 and list2 to result
    return result

Implementation

Below is a Python implementation of Strand Sort. The algorithm extracts sorted strands from the input and merges them with the output list.

Python Implementation 
def strand_sort(input_list):
    if not input_list:
        return input_list
        
    output_list = []
    
    while input_list:
        # Start a new strand with the first element
        sublist = [input_list.pop(0)]
        i = 0
        
        # Build the strand
        while i < len(input_list):
            if input_list[i] > sublist[-1]:
                sublist.append(input_list.pop(i))
            else:
                i += 1
        
        # Merge the strand with the output
        if not output_list:
            output_list = sublist
        else:
            # Merge two sorted lists
            merged = []
            while output_list and sublist:
                if output_list[0] < sublist[0]:
                    merged.append(output_list.pop(0))
                else:
                    merged.append(sublist.pop(0))
            merged.extend(output_list)
            merged.extend(sublist)
            output_list = merged
    
    return output_list

# Example usage
arr = [38, 27, 43, 3, 9, 82, 10, 15]
sorted_arr = strand_sort(arr)
print("Sorted array:", sorted_arr)
# Output: Sorted array: [3, 9, 10, 15, 27, 38, 43, 82]

Applications

Strand Sort is useful in specific scenarios:

  • Sorting linked lists efficiently
  • Educational purposes to demonstrate recursive sorting
  • When data is already partially sorted
  • Situations where stability is important
  • As an alternative to other recursive sorts

Advantages

Strand Sort has some unique characteristics:

  • Efficient for linked lists (O(n) space and O(n log n) time in best case)
  • Naturally stable algorithm
  • Performs well on partially sorted data
  • Conceptually simple recursive design
